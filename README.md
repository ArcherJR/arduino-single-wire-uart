# arduino-single-wire-uart
This project demonstrates how to transfer data between two Arduino Nano boards using a single data wire and a common ground, without any external modules.  
Bu proje, herhangi bir ek modül kullanmadan, sadece tek bir veri kablosu ve ortak şase üzerinden iki Arduino Nano arasında nasıl veri aktarılacağını gösterir.
